• Le menu fichier

    Le sous menu Ouvrir fichier permet d'ouvrir un fichier . Une fois le fichier ouvert vous pouvez naviguer dans les fichiers du dossier avec les boutons droit et gauche .

    Le sous menu Ouvrir dossier permet d'ouvrir un dossier. La première image du dossier sera affichée. Et vous pourrez naviguer ensuite dans les fichiers du dossier avec les boutons droit et gauche .

    Le sous menu Ouvrir dossier permet d'ouvrir un dossier. La première image du dossier sera affichée. Et vous pourrez naviguer ensuite dans les fichiers du dossier avec les boutons droit et gauche .

    Le sous menu Enrégistrer comme permet d'enrégistrer l'image qui s'affiche sous l'un des formats suivants: png, bmp, jpg, jpeg, ppm, xbm, xpm .

    Le sous menu imprimer permet d'imprimer l'image qui s'affiche.

  • Le menu Editer

    Le sous menu copier permet de copier l'image qui s'affiche.

    Le sous menu coller permet de coller dans le logiciel une image que vous avez copier autre part.

  • Le menu Affichage

    Le sous menu zoomer plus permet de zoomer en profondeur l'image qui s'affiche.

    Le sous menu zoomer moins permet de dézoomer l'image qui s'affiche.

    Le sous menu Ajuster à la taille de l'écran permet de d'égaliser la taille de l'image et de l'écran.

  • Le menu A propos

    Le sous menu aide permet d'afficher l'aide.

    Le sous menu Au sujet du développeur permet de s'informer sur le développeur.
